# Screenshots & Demo Assets

This directory contains screenshots and demo materials for PlantDocBot.

## Contents

- `home.png` - Main application interface
- `image-detection.png` - Image-based disease detection demo
- `text-diagnosis.png` - Text-based diagnosis demo
- `chatbot.png` - AI chatbot interface
- `results.png` - Sample prediction results

## Adding Screenshots

To add screenshots:

1. Take screenshots of the application
2. Save as PNG files with descriptive names
3. Place in this directory
4. Update README.md to reference them

## Recommended Tools

- **Windows**: Snipping Tool, Win + Shift + S
- **Mac**: Cmd + Shift + 4
- **Linux**: Screenshot tool or Flameshot

## Image Guidelines

- Format: PNG or JPG
- Max size: 2MB per image
- Resolution: 1920x1080 or similar
- Show actual functionality
- Include UI elements clearly
